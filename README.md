# Калкулатор за разход на гориво

Просто **responsive** уеб приложение за изчисляване на различни стойности, свързани с разхода на гориво на автомобил `МПС`.

Приложението е създадено с **HTML**, **CSS** и **JavaScript** и е оптимизирано основно за използване от мобилно устройство.

## Функционалности

Проектът включва три основни секции:

1. Реален разход
2. Цена на пътуване
3. Нужни литри

### 1. Реален разход

Позволява изчисляване на средния разход на автомобила на база:

- изминати километри;
- заредени литри.

Формула:

`разход = ( литри / километри ) * 100`

Резултатът се показва в: `л/100 км`

### 2. Цена на пътуване

Позволява изчисляване на приблизителната цена за дадено пътуване на база:

- разстояние в километри;
- среден разход л/100 км;
- цена на гориво за литър;
- избрана валута;
- брой пътници.

Изчисляват се:

- нужни литри гориво;
- обща цена на пътуването;
- цена на километър;
- цена на човек.

По подразбиране валутата е EUR (€).

### 3. Нужни литри

Позволява изчисляване на необходимото количество гориво за даден маршрут на база:

- разстояние;
- среден разход.

## Навигация

Приложението има навигационно меню с три бутона:

- Разход
- Цена
- Литри

При избор на секция се визуализира само съответният калкулатор, без презареждане на страницата.

## Валидация

Въведените стойности се проверяват преди изчисление. 

Приложението не позволява изчисление при:

- празни полета;
- отрицателни стойности;
- стойности, равни на 0;
- невалиден брой пътници;
- нечислови стойности.

При грешка се показва ясно съобщение към потребителя.

## Примерна употреба (Всичко е направено интуитивно, не е нужно да минаваш през тази стъпка)

### Пример за реален разход

Ако автомобилът е изминал 520 км и са заредени 42 литра, приложението ще изчисли приблизителен разход:

`8.08 л/100 км`

### Пример за цена на пътуване

Ако пътуването е 300 км, средният разход е 7.5 л/100 км, цената на горивото е 1.55 € и пътниците са 3, приложението ще изчисли:
- нужни литри;
- обща цена;
- цена на километър;
- цена на човек.
